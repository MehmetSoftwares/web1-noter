# WEB1 – Reeksamen hurtignoter (GitHub Pages)

Statisk hjemmeside til hurtig navigation under mundtlig eksamen.

## Filer
- `index.html` – alle noter (Sæt 1–12)
- `styles.css` – styling + læsbarhed (stor tekst)
- `script.js` – søgning, hurtig navigation, kopier-knapper, A-/A+ tekststørrelse
- Ingen build-tools. Bare upload.

## GitHub Pages (hurtig)
1. Opret et repo (fx `web1-noter`)
2. Upload disse filer til repoets root
3. Settings → Pages
4. Deploy from a branch → `main` + `/(root)`
5. Åbn URL’en

## Brug
- Klik et sæt → alt i sættet åbner, andre lukker
- Brug A-/A+ for tekststørrelse
- Brug søg til at finde begreber hurtigt
