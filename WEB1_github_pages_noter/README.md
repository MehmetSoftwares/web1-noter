# WEB1 – Reeksamen hurtignoter (GitHub Pages)

Dette er en statisk hjemmeside (ingen build tools), lavet til hurtig navigation under mundtlig eksamen.

## Filer
- `index.html` – alle noter (Sæt 1–12)
- `styles.css` – styling + layout
- `script.js` – hurtig navigation, søgning, copy-knapper, “skjul/vis spg. 1”

## GitHub Pages (hurtig opsætning)
1. Opret et repo på GitHub (fx `web1-noter`).
2. Upload disse 3 filer til repoets root.
3. Gå til **Settings → Pages**
4. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main** (eller master), folder: **/(root)**
5. Åbn den URL GitHub viser (typisk `https://DITBRUGERNAVN.github.io/web1-noter/`)

## Brug under eksamen
- Klik på et sæt i venstre side → hele sættet åbnes automatisk.
- Brug søgefeltet (fx “fetch”, “grid”, “box-sizing”).
- Klik “Skjul/vis Spg. 1” hvis du vil fokusere på teori-spørgsmålene.
